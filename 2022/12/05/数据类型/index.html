1. 基本数据类型（primitive type）

8大基本数据类型

  • 位（bit）：计算机内部数据储存的最小单位，习惯用b表示。例如11001100，是八位二进制数。

    1bit表示1位。

  • 字节（Byte）：计算机数据处理的基本单位。习惯用B表示。8位（bit）就称为1字节（Byte）。

    • 1Byte表示一个字节（1B）
    • 1B=8b
    • 1KB=1024B
    • 1MB=1024KB
    • 1GB=1024MB
    • 1TB=1024GB

  • 字符：指计算机中使用的数字、字母、字和符号。

2. 数据类型转换

  1. 不能对布尔值进行转换；

  2. 不能把对象类型转换为不相干的类型；

  3. 高容量转换为低容量，必须强制转换，(type)variableName；反之，不用强制。

    低容量 ------------------------------------>高容量

    byte, short, char -> int -> long -> float -> double

  4. 高容量转换为低容量，注意内存可能溢出。

  5. 操作、计算比较大的数注意可能溢出，要在计算前解决精度和数据类型的问题。

  6. 数字可以加下划线分隔，例如1亿可写为：100_000_000 或者10000_0000。
